# Nostr React client playground

## Features

- Events
- Users
  - IndexedDB storage (thousands of users)
  - Request information for unknown users in intervals
  - Follow users
- Filters

## Setup

Open up `./src/App.tsx` and explore ...

Install:

```bash
pnpm install
```

Run:

```bash
pnpm run dev
```
