# Nostr Client

Nostr web client built with React.

- Relies on IndexedDB and local storage for data and accounts
- implements `@nostr-ts/common` and `@nostr-ts/web`

- `nos2x` support is planned

## Setup

Install:

```bash
pnpm install
```

Run:

```bash
pnpm run dev
```
